# Pay Buddy

Проект на Vite + React с настроенным React Router DOM.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Использование React Router

Роутинг настроен в `src/App.jsx`. Добавьте новые маршруты в компонент `App`.

Пример использования:
- `<Link to="/path">Текст</Link>` - для навигации
- `<Route path="/path" element={<Component />} />` - для определения маршрута
